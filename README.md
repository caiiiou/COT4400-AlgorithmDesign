# COT4400 Project 2 - Algorithm Design & Analysis

Implementations of three algorithmic paradigms in C++:
- **Merge Sort** (Divide & Conquer)
- **Activity Selection** (Greedy)
- **0/1 Knapsack** (Dynamic Programming)

## Files
| File | Description |
|---|---|
| `main.cpp` | Entry point, test cases, timing |
| `merge_sort.cpp` | Merge sort implementation |
| `activity_selection.cpp` | Activity selection (greedy) |
| `knapsack.cpp` | 0/1 knapsack (DP) |

## Build & Run
```
g++ -std=c++17 -o project2 main.cpp merge_sort.cpp activity_selection.cpp knapsack.cpp
./project2
```
